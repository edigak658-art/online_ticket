BMTC Smart Digital Ticketing System
📌 Project Overview

The BMTC Smart Digital Ticketing System is a web-based mini project developed to improve the public bus ticketing process in metropolitan cities like Bengaluru.

Currently, passengers scan a QR code for payment, but conductors still issue paper tickets manually. This process consumes time, especially during rush hours in Tier-1 cities.

Our platform solves this problem by providing a fully digital ticketing solution where passengers can:

Select source and destination
Choose number and type of passengers (Adult/Child)
View bus details and route information
Make payment directly through UPI
Instantly receive a digital ticket

This system reduces manual work for conductors, saves passenger time, and promotes an eco-friendly paperless transport system.
